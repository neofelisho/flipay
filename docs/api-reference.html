    flipay v0.1.0 API Reference

    Modules

    Flipay

    Flipay is a API service provides below functions.

    Flipay.Accounts

    The Accounts context.

    Flipay.BestRateFinder

    Find the best rate according to input/output assets, input amount and exchange's order book.

    Flipay.Exchanges

    Get order book from specific exchange, according to input/output assets.

    Flipay.Exchanges.Coinbase

    Implementation of exchange: Coinbase. Currently here is a mock version.

    Flipay.Exchanges.Coinbase.Client

    WebSocket client for Coinbase. Reference the documentation of Coinbase Pro.

    Flipay.Exchanges.Coinbase.OrderBook

    Keep order book from Coinbase.

    Flipay.Exchanges.Exchange

    Defint the behaviors that each implementation of exchange should have.

    Flipay.Guardian.Plug
    FlipayWeb

    The entrypoint for defining your web interface, such as controllers, views, channels and so on.

    FlipayWeb.Authentications.AuthErrorHandler

    Authentication error handler.

    FlipayWeb.Authentications.AuthPipeline

    Authentication pipeline.

    FlipayWeb.ErrorHelpers

    Conveniences for translating and building error messages.

    FlipayWeb.ErrorView

    View for fallback controller.

    FlipayWeb.FallbackController

    Translates controller action results into valid Plug.Conn responses.

    FlipayWeb.Gettext

    A module providing Internationalization with a gettext-based API.

    FlipayWeb.QuoteController

    Handle request about asset quotes.

    FlipayWeb.QuoteView

    View for quote controller.

    FlipayWeb.Router

    Routes of Flipay API.

    FlipayWeb.Router.Helpers

    Module with named helpers generated from FlipayWeb.Router.

    FlipayWeb.UserController

    Handle request about user account and authentication.

    FlipayWeb.UserView

    View for user controller.
